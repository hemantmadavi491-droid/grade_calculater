# 🎓 Grade Calculator Using Streamlit

A modern and interactive Grade Calculator web application built with Python and Streamlit. This application allows students to enter marks for multiple subjects, calculate total marks, percentage, and automatically determine the final grade with remarks.

## 📌 Features

* Simple and user-friendly interface
* Enter marks for 5 subjects
* Automatic total marks calculation
* Percentage calculation
* Grade assignment based on percentage
* Performance remarks
* Modern UI with custom styling
* Responsive design for desktop and mobile devices
* Built using Python and Streamlit

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/grade_calculator.git
cd grade_calculator
```

### 2. Install Required Packages

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run app.py
```

If the Streamlit command is not recognized:

```bash
python -m streamlit run app.py
```

---

## 🎯 How It Works

1. Enter marks for all subjects.
2. Click the **Calculate Grade** button.
3. The application will:

   * Calculate total marks
   * Calculate percentage
   * Determine the grade
   * Display performance remarks
